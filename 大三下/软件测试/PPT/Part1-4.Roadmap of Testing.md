# Roadmap of Testing

## 测试中的概念
- Testing
    + 带着找出错误的目的去执行程序
- Debugging
    + 找出并且修复错误
- Test Input
    + 用来驱动测试的数据或操作序列
- Test Oracle
    + 用来确认测试是通过还是失败的机制
- 测试案例
    + Test Input + Test Oracle
- Test harness(测试工具)
    + 通过配置一系列的软件和测试案例来测试一个程序。

## 测试活动
- 测试设计
- 测试实施
- 测试执行
- 测试评估

## 测试技术概览(PMOD)
- Problem
    + 根本问题: 测试充分，测试预言，测试生成
    + 其他重要问题：测试自动化，测试管理
- Methods
    + Exploratory method (探索性测试方法)
    + Black box method（黑盒测试方法）
    + White box method （白盒测试方法）
    + Model-based method（基于模型的测试方法）
- Objective
- Domains
